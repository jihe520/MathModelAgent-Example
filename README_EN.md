# Introduction

This repository contains running results from the [MathModelAgent](https://github.com/jihe520/MathModelAgent) project.


# Contributing

Submit your running results via PR:

Thank you for your contributions! <‰

File submission structure:

```md
- Year Competition Name
  - Problem Number
    - questions
    - your-github-id-modeler-model-coder-model-paperwriter-model
      - Place files from your working folder here
```

Create any missing folders as needed.

Example:

```md
- 2025 National Competition
  - Problem A
    - questions
    - jihe520-deepseek-v3.1-deepseek-v3.1-deepseek-v3.1
```

```md
- 2023 Huashu Cup
  - Problem C
    - questions
    - jihe520-deepseek-v3.1-deepseek-v3.1-deepseek-v3.1
```
